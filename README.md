# Phone directory
My simple project for university
##Run
```docker-compose up```

Add ```-d``` flag for detached run

##Testing
 Swagger UI link:

 ```localhost:8080/swagger-ui.html```
 
##Tech:
 <li> Java 11
 <li> Spring boot
 <li> Swagger
 <li> h2 db
 <li> Hibernate

##Functionality

<li> CRUD for Person with filtration by name, surname, phone
<li> CRUD for Phone with filtration by all fields
